bitset笔记

发表于 2020-04-22 更新于 2020-10-01 分类于 OI笔记

bitset 即位集或者位图，是一种通过采用二进制压缩布尔型变量，基于 CPU 优化，以减少空间、提高效率的数据结构。因为 CPU 对单个变量的处理是 $O(1)$ 的，而 bitset 的内部实现采用了 C++ 中的 long 类型存储，所以对 bitset 进行左移、右移等整体操作的时间复杂度为 $O(\frac{n}{\omega})$，其中 $\omega$ 受操作系统位数决定。当前大多数操作系统为 64 位，所以此时 $\omega = 64$。
