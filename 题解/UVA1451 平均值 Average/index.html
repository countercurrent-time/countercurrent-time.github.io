UVA1451 平均值 Average

发表于 2020-02-19 更新于 2020-06-05 分类于 题解
  • UVA1451 平均值 Average

详细解析见《算法竞赛入门经典》P243。这题可以使用01分数规划或单调栈维护凸壳来做，这里写几个细节。

  1. 一看到这题，很容易产生一种错误的想法：对于每个长度为 $L$ 的区间，当 $a_{l - 1} = 1$ 或 $a_{r + 1} = 1$ 时不断向左右延展区间。Hack 也很简单：L = 5, 101000101。

  2. 刘汝佳在 P244 提到求凸壳切点的方法时讲得很不清楚。这里有一种很简单的方法：对于 $a(x, y), a’(x + 1, y + p), b(x’, y’), b’(x’ + 1, y’ + q)(p, q \in {0, 1})$，可以枚举 $p$ 和 $q$ 的四种取值进行讨论，发现在 $k_{ab’} > k_{ab}$ 时，总有 $k_{a’b’} \ge k_{a’b}$。所以每次求出的切点的 $x$ 是非严格单调递增的，因此每次枚举到点 $p$ 使对于当前点 $a$，有 $k_{a, p + 1} < k_{a, p}$ 即可，即枚举到斜率开始减小前停下来。

  3. 我之前使用了单调队列，因为求出的切点的顺序是单调的，所以看似可以把求出的切点前面的决策点进行 pop_front()，但是并不可以。因为这样会影响到之后的 pop_back() 时的判断。

  4. 觉得代码细节多的话，可以先使用 double cal(int l, int r) { return (a[r] - a[l - 1]) / (r - l + 1.0); }，在保证代码正确性后，再改为使用 check(l, r) 函数以消除浮点误差（本题使用浮点数会非常容易造成浮点误差，因为存在 $10^{-6}$ 级别的比较）。
