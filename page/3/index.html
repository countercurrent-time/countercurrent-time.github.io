贪心笔记

发表于 1970-01-01 更新于 2020-10-01 分类于 OI笔记

贪心是一种特殊的算法思想，可以在题目含有以下性质时使用：

  • 子结构最优可以推出全局最优；
  • 状态无后效性，或者后效性可以被消除。

为了消除后效性，贪心经常配合堆、排序等使用。同时贪心没有定式，题目往往非常灵活，所以更需要不错的算法思维。
