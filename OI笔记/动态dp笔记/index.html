动态dp笔记

发表于 2020-03-23 更新于 2020-05-30 分类于 OI笔记

实在讲得不够清楚，初学还是参考 GKxx 大佬的洛谷日报吧，这里权当我记录动态 dp 好题的博客了。

动态 dp 是一种题型的广泛称呼，由猫锟在 NOIP2018 中借助 D2T3 保卫王国一题广泛宣传，通常指使用数据结构实现带修改的动态规划。对于序列上的动态 dp，通常使用线段树等序列上的简单高效的数据结构；而对于树上动态 dp，则 link cut tree 应用最为广泛。

为了方便数据结构维护，我们需要使用具有结合律的，方便信息合并的状态转移方程，广义矩阵乘法也是一种比较常用的转换方法。

哪有这么多说法，动态 dp 本质就是数据结构题套了动态规划的壳

SP1716 GSS3 - Can you answer these queries III

本题要求查询区间最大子段和，由于我们平常使用的 $O(n)$ 求最大子段和不支持快速合并信息，所以我们需要考虑其他方法。

考虑到为了从两个子区间获取当前区间的最大子段和，我们需要分类讨论：可能是单独一个区间的子段和，也有可能是前一个区间的最大后缀子段和加后一个区间的最大前缀子段和。所以我们需要维护这三个信息，同时我们还需要知道区间和。

$$sum = lc_{sum} + rc_{sum};$$
$$lsum = max(lc_{lsum}, lc_{sum} + rc_{lsum});$$
$$rsum = max(rc_{rsum}, rc_{sum} + lc_{rsum});$$
$$ans = max(lc_{rsum} + rc_{lsum}, lc_{ans}, rc_{ans});$$

然后我们可以简单地使用线段树维护。

P4332 [SHOI2014]三叉神经树

本题概念较为简单，适合用于 P4719 【模板】”动态 DP”&动态树分治 的前置题目，以及 link cut tree 维护链上信息的进阶题目。本题需要充分利用题目和 link cut tree 的性质并仔细分析。

可以发现我们修改一个叶子节点的权值时会对叶子到根的路径上一段连续的，子节点权值和为 1 或 2 的节点权值进行修改，并在第一个权值非 1 或 2 的节点停下。所以我们需要知道这样的节点的位置。由于这样节点有很多，所以我们只维护我们真正需要的，离根节点最远的这样的节点的位置。

容易想到使用 link cut tree 修改并维护链上信息，而通常情况下，link cut tree 的代码更简单；对于没有换根操作的全局 link cut tree，其时间复杂度为 $O(n \log n)$。

P4719 【模板】”动态 DP”&动态树分治

本题的前置知识比较多：

  • link cut tree 以及 link cut tree 维护子树信息的方法（建议做P4219 [BJOI2014]大融合）
  • 矩阵乘法

不带修改操作时，我们使用如下状态转移方程计算每棵子树的答案：

$$f(x,0)=\sum\limits \max{f(v,0),f(v,1)}$$
$$f(x,1)=\sum\limits f(v,0)+a_x$$

此时带修改操作后，容易发现我们需要修改一条链上的状态转移信息，所以我们需要树链剖分或者 link cut tree 维护。

同时我们需要考虑快速合并子树的状态转移信息。

根据P4219 [BJOI2014]大融合中维护 link cut tree 子树信息的方法，我们需要另一个数组 $g$：

$$g(x,0)=\sum\limits_{y\neq son(x)}\max{f(y,0),f(y,1)}$$
$$g(x,1)=\sum\limits_{y\neq son(x)}f(y,0)+a_x$$

其中 $son(x)$ 是不在 $x$ 所在重链或者实链上的儿子（即轻儿子或者虚儿子）。

这样我们可以更改 $f$ 的状态转移方程：

$$f(x,0)=g(x,0)+\max{f(son(x),0),f(son(x),1)}$$
$$f(x,1)=g(x,1)+f(son(x),0)$$

然后就可以通过广义矩阵乘法快速合并子树信息。由于状态转移方程式含有 $max$ 和加法，同时 $max$ 和加法满足分配律，所以我们可以用取 $max$ 和加法代替原来的矩阵乘法的加法和乘法操作，即 $C_{i,j}=\max_k{A_{i,k}+B_{k,j}}$。所以我们可以得到下面的式子：

$$\begin{bmatrix}g(x,0)&g(x,0)\g(x,1)&-\infty\end{bmatrix}\begin{bmatrix}f(son(x),0)\f(son(x),1)\end{bmatrix}=\begin{bmatrix}f(x,0)\f(x,1)\end{bmatrix}$$

实际为了方便，我们会用 $2 \times 2$ 的矩阵代表 $f$，后面两个格子没有意义（不过直接用 $2 \times 1$ 的矩阵会更快）；同时矩阵乘法不满足交换律，要注意合并信息时的顺序。

P5024 保卫王国

本题与上一题相反，需要维护的是树上最小点覆盖集。有两种方法：

  1. 利用最小权覆盖集 = 全集 - 最大权独立集的性质，将本题转化为上一题。单点的强制选择或不选择可以分别将不选择或选择的情况的代价加上正无穷，使这种方法的代价过大。
  2. 直接采用新的矩阵和转移方程：

$$f(x, 0) = g(x, 0) + f(son(x),1)$$
$$f(x, 1) = g(x, 1) + \min{f(son(x),0),f(son(x),1)}$$

$$\begin{bmatrix}+\infty&g(x,0)\g(x,1)&g(x,1)\end{bmatrix}\begin{bmatrix}f(son(x),0)\f(son(x),1)\end{bmatrix}=\begin{bmatrix}f(x,0)\f(x,1)\end{bmatrix}$$

为了方便，动态 dp 代码实现中有 f, g 矩阵之间的复制，此时需要注意答案的位置在 $2 \times 2$ 矩阵中的位置和上一题有所不同。
